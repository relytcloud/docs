## 查看订阅情况

### 查看任务套餐订阅

1. 在管理控制台的顶部导航栏中，点击 **订阅 & 套餐**。

    <Frame>
     <img src="/admin/img/subscription-list.png" style={{ borderRadius: '0.5rem' }} />
    </Frame>


2. 在出现的订阅列表中可以查看如下信息。

    | 字段 | 说明 |
    | :- | :- |
    | 订阅 ID | 订阅的唯一标识符。 |
    | 套餐 | 订阅对应的套餐。 |
    | 生效日期 | 该订阅的生效日期。 |
    | 关联用户 | 订阅绑定的用户。如值为 **无**，表示该订阅未与任何用户绑定。可以点击该列中的展开图标，[完成与用户的绑定](#绑定订阅至用户)。 |
    | 订阅状态 | 该订阅当前的状态。|
    | 操作 | 订阅支持的操作。|


---

## 绑定订阅至用户

同一时间内，一个订阅只能绑定至一个用户。绑定后，用户可以使用该订阅对应的套餐里的任务配额。

<Warning>
 只有绑定了订阅的用户才能执行数据分析和探索任务。
</Warning>

1. 在管理控制台的顶部导航栏中，点击 **订阅 & 套餐**。

    <Frame>
     <img src="/admin/img/subscription-list.png" style={{ borderRadius: '0.5rem' }} />
    </Frame>

2. 找到目标订阅，点击 **关联用户** 列中的展开图标，在弹出的 **关联用户至套餐** 的对话框中，选择目标用户的 ID，点击 **提交**。

    <Frame>
     <img src="/admin/img/associate-user.png" style={{ borderRadius: '0.5rem' }} />
    </Frame>
